# Chess_Game
## --- using C++ and Qt<br>------ A Mult-player Online Chess Game
# How to compile

First,make sure you've installed the qt5<br>
Using the command to compile
```bash
git clone https://github.com/haoxingxing/Chess_Game
cd Chess_Game
# Make the client
cd client
qmake client.pro
make
cd ..
# Make the server
cd server
qmake server.pro
make
cd ..
```
